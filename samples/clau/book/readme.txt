This disk contains the source code for all the examples in the book. 
The directories correspond to the chapters in the book.

The source code has been tested using the SOM Toolkit 2.0 with
CSD201 and the IBM CSet 2.1. Makefiles are provided in each
directory to build the corresponding programs.
